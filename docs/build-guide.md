This guide explains how to build cosmic-os from source.

## Requirements
- Ubuntu/Debian host
- x86_64 system
- 8GB RAM recommended

## Steps
1. Install build dependencies
2. Compile Linux kernel
3. Build BusyBox
4. Create root filesystem
5. Build initramfs
6. Test with QEMU
7. Generate Live ISO
